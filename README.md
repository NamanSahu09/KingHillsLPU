# 📖 Overview

KINGHILLSLPU was built with the idea of solving everyday campus-related problems that students face regularly.  

Instead of switching between multiple apps or searching manually for locations and services, this app aims to bring everything together into one smooth Android experience.

The project also served as a hands-on learning journey into Android development, API integration, UI/UX design, and real-world mobile application architecture.

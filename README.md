KINGHILLSLPU

A smart campus companion app built for Lovely Professional University students.
KINGHILLSLPU was created with the idea of making everyday student activities simpler, faster, and more connected inside the campus ecosystem.

From navigation support to student utilities, this app aims to provide a smoother university experience through a clean Android application.

About The Project:

KINGHILLSLPU is an Android application developed using Java and Kotlin that focuses on solving common student problems inside the university campus.

The project was built as a practical learning experience while exploring Android development, Google Maps integration, APIs, UI/UX design, and mobile app architecture.

The application includes campus-related utilities and smart features designed specifically for students.

Features
-Campus Navigation using Google Maps API
- Location-based assistance
- Clean and user-friendly UI
- Fast and lightweight Android experience
- Student-focused utilities and functionalities
- Easy access to important campus locations
- Real-time interaction with map services
- Tech Stack

Getting Started
Prerequisites

Before running the project, make sure you have:

Android Studio installed
Android SDK configured
Google Maps API Key
Java JDK installed
⚙️ Installation
Clone the repository
git clone https://github.com/NamanSahu09/KingHillsLPU.git
Open the project in Android Studio
Sync Gradle files
Add your Google Maps API Key inside:
AndroidManifest.xml
Run the application on Emulator or Physical Device

Google Maps Setup

To enable maps functionality:

Create a project in Google Cloud Console
Enable Maps SDK for Android
Generate an API Key
Add the key inside the manifest file

Example:

<meta-data
    android:name="com.google.android.geo.API_KEY"
    android:value="YOUR_API_KEY"/>
